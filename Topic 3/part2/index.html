CST-120 Web Development Syllabus

Course Description

This course introduces the fundamental concepts and syntax of web development languages including HTML, CSS, and JavaScript. The course focuses on foundation required to build complex dynamic web applications. Prerequisite: none.

Weeks 1-2

Topic 1 HTML

In this topic students are introduced to the HTML markup language where they learn basic structure, basic elements of an HTML document, and how to work with text and basic controls. The students finish this topic by learning the developer tools available to them, as well as how web servers work to help support development of HTML-base applications.

  1. Develop webpages with basic elements and structure that include, but are not limited to, document, head, and body tags.
  2. Develop webpages with text that includes, but is not limited to, headings, paragraphs, formatting, fonts, and color tags.
  3. Develop webpages with components that include, but are not limited to, links, images, and lists tags.
  4. Investigate how a web server functions. Identify differences between localhost, file URL and domain names in the URL.
  5. Investigate the developer's tools available as a webpage developer.

Readings

      W3Schools: Go to https://www.w3schools.com. Select the Learn HTML menu under the HTML and CSS section. Complete all reading and tutorials under the HTML Home Tutorial section.

Weeks 3-4

Topic 2 Advanced HTML

In this topic, students continue to expand their knowledge of the HTML markup language, learning some of the more advanced features of the HTML markup language and how to work with layouts, graphics, and media content. The students finish this topic by learning how to make HTML pages ADA compliant.

  1. Develop webpages with layouts that include, but are not limited to, blocks/div/span, tables, and layouts tags.
  2. Develop webpages with Graphics that include, but are not limited to, image, canvas, and SVG tags.
  3. Develop webpages with media that include, but are not limited to, video and audio tags.
  4. Explain how and why ADA compliance is achieved with ADA guidelines.

Readings

      W3Schools: Go to https://www.w3schools.com. Select the Learn HTML menu under the HTML and CSS section. Complete all reading and tutorials under the HTML Forms, HTML Graphics, HTML Media, and HTML API's Tutorial sections.

Weeks 5-6

Topic 3 CSS

In this topic, students learn Cascading Style Sheets (CSS) and to apply styling to HTML pages using fonts, colors, and formatting. The students finish the topic by learning how to apply backgrounds, transitions, and animations to their HTML pages.

  1. Develop webpages with CSS basics styles that include, but are not limited to, inline, internal, external, and selectors styles.
  2. Develop webpages with text that include, but are not limited to, built-in fonts, custom fonts from external sources, color, and alignment styles.
  3. Develop webpages with formatting and layouts that include, but are not limited to, backgrounds, borders, margins, padding, width/height, and the box model styles.
  4. Develop webpages that include, but are not limited to, transitions and animations styles.

Readings

      W3Schools: Go to https://www.w3schools.com. Select the Learn CSS menu under the HTML and CSS section. Complete all reading and tutorials under the CSS Tutorial and CSS Advanced Tutorial sections.

Weeks 7-8

Topic 4 JavaScript

In this topic, students are introduced to the JavaScript programming language. This programming language is used as the foundation for client-side programming on webpages. Students learn the basic language syntax, how to work with variables, make decisions, perform repetitive operations, and create reusable functions.

  1. Develop code that integrates basic JavaScript (inline and external files) in a webpage.
  2. Develop code that leverages variables and data types in a webpage.
  3. Develop code that leverages operators (math, assignment, comparison, logical, Boolean, and string) in a webpage.
  4. Develop code that uses controls (if, for, loops) in a webpage.
  5. Develop code that uses reusable functions in a webpage.

Readings

      W3Schools: Go to https://www.w3schools.com. Select the Learn JavaScript menu under the JavaScript section. Complete all reading and tutorials under the JS Tutorial Tutorial section.

Weeks 9-10

Topic 5 Advanced JavaScript

In this topic, students continue to expand their knowledge of the JavaScript programming language, learning how to create objects, work with data as arrays, and handle events from DOM objects from their HTML pages. The students finish the topic by also being introduced to the JavaScript Object Notation (JSON), learning the advanced features of newer Page 9 of 15 updates to the JavaScript programming language and how this programming language is used along with current web application frameworks as the foundation for building modern dynamic web applications.

  1. Develop code that leverages objects in a webpage.
  2. Develop code that leverages arrays in a webpage.
  3. Develop code that handles events in a webpage.
  4. Explain what JSON is and how this data format is used to build web applications.
  5. Investigate advanced features found in recent updates to the JavaScript programming language.
  6. Investigate recent JavaScript web application frameworks that are available to build dynamic web applications.

Readings

      W3Schools: Go to https://www.w3schools.com. Select the Learn JavaScript menu under the JavaScript section. Complete all reading and tutorials under the JS Forms, JS Objects, JS Functions, and JS HTML DOM Tutorial sections.
      W3Schools: Go to https://www.w3schools.com. Select the Learn JavaScript menu under the JavaScript section. Complete all reading and tutorials under the JS Browser BOM, JS AJAX, and JS JSON Tutorial sections.

Weeks 11-13

Topic 6 Web Development Libraries

In this topic, students learn to use web development libraries to add dynamic, rich, and compelling content and user interfaces to their webpages. Students are also introduced to the concept of responsive webpages, what this means, and how they can support this concept when building their webpages.

  1. Develop webpages that supports a responsive design.
  2. Develop webpages that leverage the use of jQuery, which includes but is not limited to, the use of selectors, events, DOM manipulation, AJAX, and data table features.
  3. Develop webpages that leverage the use of Bootstrap, which includes but is not limited to, the use of the grid system and various UI components (buttons, text, alerts, tables, forms, and NavBar).

Readings

      W3Schools: Go to https://www.w3schools.com. Select the Learn CSS menu under the HTML and CSS section. Complete all reading and tutorials under the CSS Responsive and CSS Grid Tutorial sections.
      W3Schools: Go to https://www.w3schools.com. Select the Learn jQuery menu under the JavaScript section. Complete all reading and tutorials under the jQuery Tutorial, jQuery Effects, jQuery HTML, and jQuery AJAX Tutorial sections.
      W3Schools : Go to https://www.w3schools.com. Select the Learn Bootstrap menu under the HTML and CSS section. Complete all reading and tutorials for the latest version of Bootstrap under the Bootstrap Tutorial and Bootstrap Grid Tutorial sections.

Weeks 14-15

Topic 7 Advanced Web Development

In this topic, students learn that building static webpages has limits and that they often need to be deployed on web servers sometimes backed with a server-side programming language or database. This topic, at minimum, will introduce the students to the HTTP protocol, client server model, HTML forms, and how to implement forms processing to expand on the capabilities of the webpages they have built so far in previous topics. This topic introduces the students to the basic concepts required for future web application development courses where formal dynamic web application frameworks are used.

  1. Investigate the HTTP protocol and client server model.
  2. Develop webpages with HTML forms.
  3. Develop webpages that integrate forms processing on the server.

Readings

      None
